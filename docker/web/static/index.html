El objetivo de este TFM es simular el funcionamiento de las API de Open Gateway en un entorno controlado. Esta web permite gestionar la base de clientes del proyecto así como probar las API disponibles.

Por favor, para gestionar la base de clientes haga click en el botón de provision.

Para probar las API disponibles, haga click en el botón de sandbox.
